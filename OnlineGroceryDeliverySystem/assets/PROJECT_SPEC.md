# Online Grocery Delivery System

> Production-ready MERN Stack Project

---

# 1. Project Overview

This project is a full-stack Online Grocery Delivery System built using the MERN Stack.

The application allows customers to browse grocery products, search products, add them to a shopping cart, place orders, make online payments, track deliveries, and manage their profiles.

The application also provides a complete Admin Dashboard to manage products, categories, inventory, orders, customers, delivery partners, offers, coupons, and reports.

The system should be scalable, modular, secure, responsive, and production-ready.
